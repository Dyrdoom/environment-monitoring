# 🖥️ Клієнтська частина (Frontend) - Eco Monitor System

Клієнтська частина – це односторінковий додаток, створений за допомогою **React** та **Vite**. Використовує **Axios** для взаємодії з Backend API.

## Основні функціональні можливості

1.  **Навігація:** Легке перемикання між модулями "Моніторинг якості повітря" та "Розрахунок збитків".
2.  **Моніторинг:** Відображення табличних даних моніторингу з можливістю фільтрації.
3.  **Модуль розрахунку збитків:** Інтерактивний інтерфейс для:
    * Виконання розрахунку за формулою ($M_{excess} \times Base \times K...$).
    * Генерації синтетичних даних для тестування.
    * Редагування конфігурованих базових ставок через UI.
