# 💻 Серверна частина (Backend) - Eco Monitor API

Серверна частина побудована на базі Node.js та Express з використанням MongoDB (через Mongoose) для керування даними.

## Основні функціональні можливості

1.  **Моніторинг якості повітря (AQ):** API для отримання, додавання та керування даними про якість повітря.
2.  **Модуль розрахунку екологічних збитків:** API для розрахунку економічних втрат від наднормативних викидів. Підтримує генерацію синтетичних даних та конфігуровані коефіцієнти.
3.  **Автоматичний збір даних:** Заплановане завдання для періодичного оновлення даних про якість повітря через зовнішні API.
